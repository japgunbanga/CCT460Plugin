== Description ==
 
Easily create and manage your cake recipes for your bakery website in a central location.
 
= Plugin Features =
 
*   Manage all your cake recipes using a custom post type.
*   Include cake reipe documentation on any page using a simple shortcode.
*   Change Background color of the plugin to white or pink using the settings button in the wordpress dashboard 

 
== Installation ==
 
1.  Upload your plugin folder to the 'wp/content/plugins' directory.
2.  Activate the plugin through the 'Plugins' menu in WordPress.
3.  Scroll down to 'Custom Post Type' plugin 
4.  Click 'Activiate' 
 
 Now your ready to go!
